Majestic style система заявок

Что внутри:
- красивые заявки как на Majestic
- кнопки: Принять / На рассмотрении / Отклонить
- красивый лог принятия
- красивый лог отказа
- статус заявки в embed
- выдача стартовой роли при принятии
- оптимизированная панель семьи без лишнего спама API

Важно:
- укажи ACCESS_APPLICATIONS через запятую
- укажи APPLICATION_DEFAULT_ROLE
- не ставь UPDATE_INTERVAL_MS ниже 60000
