Family Bot v2

Что умеет:
- красивые embed-списки
- онлайн/оффлайн статусы Discord
- автообновление каждые X секунд

Важно:
1. В Discord Developer Portal включи:
   - SERVER MEMBERS INTENT
   - PRESENCE INTENT

2. В Railway Variables добавь:
   TOKEN
   GUILD_ID
   CHANNEL_ID
   MESSAGE_ID
   ROLE_LEADER
   ROLE_DEPUTY
   ROLE_ELDER
   ROLE_MEMBER
   ROLE_NEWBIE
   UPDATE_INTERVAL_MS

3. Если бот создал новое сообщение, скопируй его ID в MESSAGE_ID.

Статусы:
🟢 online
🟡 idle
⛔ dnd
⚫ offline
